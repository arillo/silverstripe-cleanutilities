Silverstripe-cleanutilities

Arillo CleanUtilities is a module for Silverstripe 2.4.x which provides functionality that makes the daily development easier and faster. It comes with a couple of classes which can be easily added as extensions to the SiteTree class. It also gives more functionality to several other sapphire classes. Arillo CleanUtlities

Arillo CleanUtilities is a module for Silverstripe 2.4.x which provides a bunch of functionality that makes the daily development easier and faster. It comes with a couple of asset classes which can be easily added to SiteTree classes in a has-many fashion.

It also gives you more functionality to several other sapphire classes. For more information, please consult the docs and the example sections.

Installation

As with all Silverstripe modules, you have to copy cleanutilities folder and all dependent modules into your project's root directory. After that you should run dev/build and you are done.

Dependencies

CleanUtilities is dependend on following third-party modules:

  • Uncle Cheese´s DataObjectManager
  • Uncle Cheese´s Uploadify
